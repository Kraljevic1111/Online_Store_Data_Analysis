# Order Analytics Project

## 📌 Overview
This project demonstrates how to use **Pandas DataFrames** to analyze data about orders, customers, and products. It merges multiple tables and calculates key sales and revenue metrics.

## ⚙️ Features
- Merge tables (`orders`, `customers`, `products`) into a single DataFrame
- Calculate total revenue per customer
- Identify the **best customer** (highest revenue)
- Calculate total revenue per product
- Identify the **best product** (highest revenue)
- Find the **most sold product by quantity**
- Analyze revenue by city
- Parse dates and analyze sales trends by date
- Find the best customer in each city

## 🛠️ Technologies
- Python 3
- Pandas library
